Resumen

Este proyecto consiste en el desarrollo de una simulación computacional en Python para analizar el comportamiento de un circuito RLC serie en corriente alterna (AC). El objetivo es construir una herramienta que permita visualizar gráficamente cómo varían la impedancia y la corriente del circuito al modificar la frecuencia de la fuente, identificando especialmente la frecuencia de resonancia.

El proyecto integra conceptos de electromagnetismo (Física II) con habilidades de programación propias de Ingeniería de Sistemas, generando gráficas interactivas sin necesidad de herramientas comerciales como MATLAB o Multisim.

Etapa 1 — Recopilación de información

Marco teórico

Un circuito RLC en serie está compuesto por una resistencia R, un inductor L y un capacitor C conectados en serie a una fuente de voltaje alterno. En corriente alterna, cada elemento presenta una oposición al flujo de corriente denominada reactancia:

  • Reactancia inductiva: XL = ωL = 2πfL
  • Reactancia capacitiva: XC = 1 / (ωC) = 1 / (2πfC)
  • Impedancia total: Z = √(R² + (XL – XC)²)
  • Corriente: I = V₀ / Z

La frecuencia de resonancia ocurre cuando XL = XC, lo que hace que Z = R (mínimo) y la corriente sea máxima: f0 = 1 / (2π√(LC))

Justificación

El análisis teórico de circuitos RLC involucra modelos matemáticos complejos que resultan abstractos sin una herramienta visual. Una simulación computacional permite observar dinámicamente cómo responde el sistema ante diferentes frecuencias, facilitando la comprensión de fenómenos como la resonancia, el desfase y la variación de impedancia.

Adicionalmente, el desarrollo de la herramienta en Python promueve la integración entre el conocimiento físico y las competencias computacionales del estudiante de Ingeniería de Sistemas.

Código Python — Simulación RLC

simulacion_rlc.py 
import numpy as np
import matplotlib.pyplot as plt

# ── Parámetros del circuito ──────────────────────────────
R = 100        # Resistencia en Ohms
L = 0.1        # Inductancia en Henrios
C = 1e-6       # Capacitancia en Faradios
V0 = 10        # Voltaje pico de la fuente (V)

# ── Rango de frecuencias (Hz) ─────────────────────────────
f = np.linspace(1, 10000, 5000)
omega = 2 * np.pi * f

# ── Cálculo de reactancias e impedancia ──────────────────
XL = omega * L
XC = 1 / (omega * C)
Z  = np.sqrt(R**2 + (XL - XC)**2)
I  = V0 / Z

# ── Frecuencia de resonancia ─────────────────────────────
f0 = 1 / (2 * np.pi * np.sqrt(L * C))
print(f"Frecuencia de resonancia: {f0:.2f} Hz")

# ── Gráficas ─────────────────────────────────────────────
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(9, 7))
fig.suptitle("Simulación Circuito RLC Serie", fontsize=14)

ax1.plot(f, Z, color="#534AB7", linewidth=2)
ax1.axvline(f0, color="#D85A30", linestyle="--", label=f"f₀ = {f0:.1f} Hz")
ax1.set_xlabel("Frecuencia (Hz)")
ax1.set_ylabel("Impedancia Z (Ω)")
ax1.set_title("Impedancia vs. Frecuencia")
ax1.legend(); ax1.grid(alpha=0.3)

ax2.plot(f, I * 1000, color="#0F6E56", linewidth=2)
ax2.axvline(f0, color="#D85A30", linestyle="--", label=f"f₀ = {f0:.1f} Hz")
ax2.set_xlabel("Frecuencia (Hz)")
ax2.set_ylabel("Corriente I (mA)")
ax2.set_title("Corriente vs. Frecuencia (resonancia = máximo)")
ax2.legend(); ax2.grid(alpha=0.3)

plt.tight_layout()
plt.savefig("grafica_rlc.png", dpi=150)
plt.show()

Dificultades encontradas y soluciones

Problema: La división entre frecuencias muy bajas generaba valores de XC extremadamente altos, distorsionando la escala de las gráficas.
Solución: Se limitó el rango de frecuencias a un mínimo de 1 Hz y se usó escala logarítmica en el eje X para mejor visualización.

Problema: Instalación de Matplotlib en el entorno de desarrollo.
Solución: Uso de entorno virtual con pip install numpy matplotlib.
